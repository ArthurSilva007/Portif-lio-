StockSystemV2

Sistema de Gerenciamento de Estoque para Oficinas Comerciais

O Problema

Oficinas comerciais perdem tempo e dinheiro por não terem controle preciso do estoque de peças e materiais, usando planilhas desatualizadas e sem rastreabilidade.

A Solução

API REST robusta desenvolvida com Java 21 e Spring Boot 4.0.3, utilizando Spring MVC para exposição dos endpoints, Spring Data JPA para persistência, Spring Validation para garantia da integridade dos dados e PostgreSQL como banco relacional. Conta com testes unitários e de integração usando Spring Boot Test, Spring Validation Test e Spring WebMVC Test.

Java 21 Spring Boot 4.0.3 Spring MVC Spring Data JPA Spring Validation PostgreSQL Lombok Maven JUnit

Automação de Mensagens

Bot de Promoções com Spring Boot e Telegram

O Problema

Campanhas de promoções exigem comunicação rápida e em massa com clientes. Fazer isso manualmente em grupos de mensageria consome tempo, é propenso a erros e dificulta o controle e rastreabilidade dos disparos.

A Solução

Backend robusto desenvolvido com Java 17 e Spring Boot 3, responsável pela automação de disparos e gerenciamento de grupos via Telegram Bot API. Utiliza Spring Scheduler para execução programada, Spring Data JPA com PostgreSQL para persistência, Spring WebClient para integrações HTTP e MapStruct para mapeamento DTO/Entity. Testado com JUnit 5, Mockito, Testcontainers e WireMock, garantindo qualidade e confiabilidade.

Java 17 Spring Boot 3 Spring Data JPA PostgreSQL Spring Scheduler WebClient Telegram Bot API JUnit 5 Mockito Testcontainers WireMock Maven

Tasks_focus

Sistema de Tarefas com Arquitetura em Camadas e Segurança JWT

O Problema

Aplicativos de tarefas simples frequentemente ignoram a segurança dos dados, não oferecem controle de acesso adequado e carecem de visibilidade sobre produtividade e métricas.

A Solução

Backend estruturado em arquitetura em camadas com Java e Spring Boot, separado em módulos de Autenticação, Tarefas e Dashboard. Segurança implementada com JWT Token e filtros de validação, persistência com JPA/Hibernate e PostgreSQL, notificações por e-mail via MailerSend, agendamento com Quartz Scheduler e monitoramento através do Spring Boot Actuator.

Java Spring Boot Spring Security JWT JPA/Hibernate PostgreSQL Spring Mail Quartz Actuator REST API Angular
